Laravel Telegram Feedback & Email System

Thank you for purchasing this script! Below are the setup instructions and usage guide.

üì¶ Installation

  1. Unzip the project into your Laravel folder.
  2. Run composer install in terminal.
  3. Copy .env.example to .env and configure your database and mail settings.
  4. Run php artisan migrate to create the required tables.
  5. Set your Telegram bot webhook:
    php artisan telegraph:set-webhook {your_bot_id}
  6. Launch the application:
    php artisan serve
  7. Access: http://127.0.0.1:8000

ü§ñ Telegram Bot Setup

  • Create a bot via BotFather.
  • Copy the token and paste it into the admin panel form.
  • To receive messages in a group, add your bot to the group and send any message.
  • After that, your group chat_id will be captured automatically.
  • Make sure your bot has admin rights in the group.

üìß Email Notification

  • Set your SMTP details in .env file:
	
APP_URL=https://example.domain	
	
MAIL_MAILER=smtp
MAIL_HOST=smtp.mailtrap.io
MAIL_PORT=2525
MAIL_USERNAME=your_username
MAIL_PASSWORD=your_password
MAIL_FROM_ADDRESS=admin@example.com
MAIL_FROM_NAME="JobResume Bot"

MY_GROUP_ID=-1001234567890
MY_EMAIL=admin@example.com

üîê Security Notes

  • Bot tokens are masked in the admin interface.
  • Email is only sent to addresses defined in your .env file.

üìå Requirements

  • SSL enabled domain
  • Laravel 10 or 11
  • PHP 8.1+
  • MySQL or PostgreSQL database
  • Valid Telegram bot token
